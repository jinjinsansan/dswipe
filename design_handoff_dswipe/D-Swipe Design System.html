Momentum

一貫したライト基調に、紺（navy）のクロームとシアン→アクアのグラデーションを重ねた、信頼感と勢いを両立するフィンテック系デザインシステム。Tailwind v4 のトークンとして実装します。

09 — Implementation

Tailwind v4 トークン

app/globals.css の @theme に下記を定義すると、bg-brand / text-ink / rounded-card などのユーティリティが生成されます。既存の API・型・ルーティング・ストアは変更しません。

/* app/globals.css */ @import "tailwindcss"; @theme { /* brand */ --color-brand: #0284c7; --color-brand-strong: #0369a1; --color-cyan-400: #22d3ee; --color-cyan-500: #06b6d4; --color-sky-500: #0ea5e9; /* navy chrome */ --color-navy-900: #0b1f3a; --color-navy-800: #122c4d; --color-navy-700: #1b3a61; /* neutrals */ --color-ink: #0b1f3a; --color-text: #1f2c3d; --color-muted: #64748b; --color-line: #e2ebf6; --color-canvas: #f4f8fd; --color-surface: #ffffff; --color-surface-tint: #e9f6fe; /* semantic */ --color-success: #16a34a; --color-warning: #d97706; --color-danger: #dc2626; /* radius */ --radius-md: 12px; --radius-lg: 13px; --radius-card: 20px; /* type */ --font-sans: "Noto Sans JP", system-ui, sans-serif; } /* gradient + glow utilities */ @utility bg-brand-grad { background-image: linear-gradient(135deg, #0ea5e9, #06b6d4); } @utility shadow-glow { box-shadow: 0 10px 26px -8px rgba(6,182,212,.55); }

この後、選択いただいたこのシステムで トップ（/） と 公開LPビュー から高精細モックを作成します。
